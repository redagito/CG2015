Renderer features
* Modern OpenGL (3.3+ core profile)
* Deferred renderer
* Shadow mapping
* Omni-directional shadow mapping
* HDR rendering with tonemapping
* Bloom
* Depth of field (Gauss blur based)
* FXAA
* Fog
* God ray
* Vignette with distortion/blur effect
